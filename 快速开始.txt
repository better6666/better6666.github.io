====================================
  Claude AI 克隆 - 快速开始指南
====================================

🎉 恭喜！您现在拥有一个完整的 Claude AI 克隆！

📋 第一步：安装 Node.js
--------------------------
如果还没有安装 Node.js：
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本
3. 安装完成后重启命令行

📋 第二步：获取 API Key
--------------------------
1. 访问 https://console.anthropic.com/settings/keys
2. 注册/登录 Anthropic 账户
3. 点击 "Create Key" 创建新的 API Key
4. 复制 API Key（格式：sk-ant-api03-xxx）

📋 第三步：设置 API Key
--------------------------
方法 A - 使用环境变量（推荐）：
打开 PowerShell 或 CMD，运行：
  
  set ANTHROPIC_API_KEY=sk-ant-api03-你的API密钥

方法 B - 直接修改代码：
打开 server.js 文件，找到第 13 行：
  
  apiKey: 'your-api-key-here',
  
替换为您的实际 API Key

📋 第四步：启动服务器
--------------------------
双击运行：start.bat

或手动运行：
1. 打开命令行
2. 进入项目目录
3. 运行：npm install
4. 运行：npm start

📋 第五步：开始使用！
--------------------------
1. 打开浏览器
2. 访问 http://localhost:3000
3. 开始与 Claude 对话！

🔧 常见问题
--------------------------
Q: 提示 "未检测到 Node.js"？
A: 请先安装 Node.js 并重启命令行

Q: 提示 API 错误？
A: 检查 API Key 是否正确设置

Q: 服务器启动失败？
A: 确保 3000 端口未被占用

Q: API 调用超时？
A: 检查网络连接，可能需要代理
