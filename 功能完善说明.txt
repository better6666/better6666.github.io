========================================
技术实现：
========================================

**HTML 结构：**
- 为 Get help 添加了子菜单容器
- 为 Learn more 添加了子菜单容器
- 所有子菜单使用相同的 CSS 类名 `.language-submenu`

**CSS 样式：**
```css
.input-section {
    margin: 0 auto;  /* 居中显示 */
}
```

**JavaScript 交互：**
```javascript
// Get help 子菜单
getHelpBtn.addEventListener('mouseenter', () => {
    getHelpMenu.style.display = 'block';
});

// Learn more 子菜单
learnMoreBtn.addEventListener('mouseenter', () => {
    learnMoreMenu.style.display = 'block';
});
```
