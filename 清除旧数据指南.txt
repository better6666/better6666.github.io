🔄 清除旧数据并重新开始

========================================
问题：左侧栏显示旧的对话记录
========================================

原因：之前的对话存储在 localStorage 中
解决：清除浏览器缓存

========================================
操作步骤（重要！）
========================================

步骤1：清除浏览器数据
-------------------------

方法A：使用开发者工具（推荐）
1. 按 F12 打开开发者工具
2. 切换到 "Console" 标签
3. 粘贴以下代码并回车：

   localStorage.clear()
   sessionStorage.clear()
   location.reload()

4. 页面会自动刷新

方法B：手动清除
1. 按 F12 打开开发者工具
2. 切换到 "Application" 标签
3. 左侧找到 "Local Storage" → 展开 → 点击 localhost:3000
4. 右键 → "Clear"
5. 左侧找到 "Session Storage" → 展开 → 点击 localhost:3000
6. 右键 → "Clear"
7. 按 Ctrl + Shift + R 强制刷新

步骤2：选择正确的模型
-------------------------

✅ 选择：🌙 Kimi AI
   - 已配置 API Key
   - 可以正常使用

❌ 不要选择：Sonnet 4.5（Claude）
   - 未配置 API Key
   - 会导致服务器崩溃

步骤3：验证效果
-------------------------

✅ 左侧栏应该是空的（没有任何历史记录）
✅ 输入框在中间（欢迎界面）
✅ 发送消息后，输入框移到底部（正常行为）

========================================
关于输入框位置的说明
========================================

这是正常的设计：

1. 欢迎界面（首次访问）
   → 输入框在中间
   → 显示"Hey there, yuqingwei"
   → 显示快捷按钮（Code, Write, Learn等）

2. 对话界面（发送消息后）
   → 输入框移到底部
   → 显示对话内容
   → 左侧栏出现对话记录

这与官方 Claude 网站的行为完全一致！
