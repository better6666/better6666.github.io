✅ 紧急修复：欢迎界面空白问题

========================================
问题现象：
========================================

打开 chat.html 后：
❌ 中间区域完全空白
❌ 看不到 "Hey there, yuqingwei" 标题
❌ 看不到快捷按钮
✅ 左侧边栏正常
✅ 输入框在底部显示

========================================
问题原因：
========================================

CSS 布局问题：
1. `.input-section` 缺少 `margin: 0 auto` 导致不居中
2. `.welcome-section` 缺少 `align-items: center` 导致内容不居中
3. `.welcome-section` 缺少 `min-height: 0` 导致 flex 布局计算错误

========================================
修复内容：
========================================

1. ✅ 恢复 `.input-section` 的居中样式：
   ```css
   .input-section {
       margin: 0 auto;  /* 水平居中 */
   }
   ```

2. ✅ 完善 `.welcome-section` 的布局：
   ```css
   .welcome-section {
       align-items: center;  /* 子元素水平居中 */
       min-height: 0;        /* 修复 flex 计算 */
   }
   ```
